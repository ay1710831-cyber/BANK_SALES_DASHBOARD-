🏦 Bank Dashboard (Power BI)
📌 Project Overview
This project is a Power BI dashboard designed to analyze banking data and provide insights into financial performance, customer behavior, and operational trends. It helps in monitoring key metrics such as loans, deposits, and interest rates for better decision-making.

🚀 Features
💰 Total Deposits & Loan Analysis
👥 Customer Segmentation Insights
📊 Interest Rate & Financial Performance Tracking
🌍 Region-wise Banking Analysis
📅 Time-based Trends (Monthly/Yearly)
🎛️ Interactive Filters & Slicers

🔍 Key Features:
Analysis of total deposits and loan amounts
Customer segmentation and distribution insights
Interest rate and financial performance tracking
Region-wise banking performance
Interactive filters and slicers for dynamic analysis

🎯 Objective:
The goal of this project is to demonstrate:
Data visualization using Power BI
Financial data analysis and reporting
Use of DAX for calculations
Building interactive and user-friendly dashboards

📊 Key Insights
Identifies distribution of loans and deposits
Highlights high-value customers
Shows financial trends over time
Analyzes regional performance differences
