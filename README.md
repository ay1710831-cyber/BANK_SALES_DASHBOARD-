📊 Bank Loan Report | Summary Dashboard
📌 Overview

This project presents a comprehensive Bank Loan Analysis Dashboard built using Power BI. It provides clear insights into loan distribution, borrower profiles, repayment behavior, and overall financial performance through interactive visualizations.

🎯 Objectives
Analyze loan distribution and trends
Understand borrower demographics
Monitor loan performance and risk
Support data-driven decision-making

📈 Key Features
Loan Summary Metrics
Total Loan Amount
Total Applications
Interest Earned
Average Interest Rate
Borrower Analysis
Gender-wise distribution
Employment status
Home ownership
Loan Performance
Fully Paid vs Charged-Off loans
Risk and default analysis
Time-Based Insights
Monthly trends
Month-to-Date (MTD)
Year-to-Date (YTD)
Geographical Analysis
Region/state-wise loan distribution

💡 Insights
Identifies high-risk loan segments
Tracks lending and repayment trends
Helps improve financial decision-making

📌 Conclusion
This dashboard provides a complete view of loan operations, enabling better analysis, monitoring, and strategic planning.
